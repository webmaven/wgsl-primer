# Arrays Overview

Arrays are indexed collections of elements.

The first element is at index 0.

Arrays either have a fixed size: <code>array&lt;<span class="template template-array-t">T</span>, <span class="template template-array-n">N</span>&gt;</code>
or are runtime-sized <code>array&lt;<span class="template template-array-t">T</span>&gt;</code>.
