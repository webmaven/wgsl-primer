# Functions Overview

Functions in WGSL are named, reusable subroutines. Learn how to write and use them:

- **[Syntax](syntax.md)**: Explore the syntax for function parameters, return types, and bodies.
- **[Calls](calls.md)**: Understand how functions are called and how arguments are passed (including by-value vs pointers).
- **[Entry Points](entry-points.md)**: Learn how shader entry points are defined for vertex, fragment, and compute pipelines using attributes like `@vertex`, `@fragment`, and `@compute`.
- **[Must Use Attributes](must-use.md)**: Ensure function return values are not ignored using `@must_use`.
